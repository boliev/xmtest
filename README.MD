# Introduction
This solution contains two separate parts backend and frontend. Both parts work via docker.

Frontend part:
 - Nginx
 - front-end framework: materializecss (https://materializecss.com)
 - JS: jQuery
 - charts: Chart.js (https://www.chartjs.org/) 
 
Backend part:
 - PHP 7.4
 - Symfony 5.1
 - Nginx
 - Redis
